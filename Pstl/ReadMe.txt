blasonToSvg_v1.8

Description de blason , de xml vers svg 
contient les sources , un dossier samplesXML contenant des blasons décrit en XML et une librairie de meubles 
dans le dossier imagesSvg .
Installation :
Aprés l'import du projet, il faut ajouter au path du programme les jar "batik-1.8.jar" et "jdom-2.0.6.jar" 
contenu dans les dossier "batik-1.8" et "jdom" 
Utilisation : 
Décrire un blason sous format XML selon le tuto que vous pouvez trouvez dans le fichier Manuel d'utilisation , et le placer 
dans le dossier sampleXML Lancer le programme main qui ce trouve dans la classe Heraldique du package execution 
Le resultat des samples ce trouvera dans le dossier blasonSVG
